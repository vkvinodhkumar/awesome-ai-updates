# AI Terminology

Last Updated: 2026-07-19 08:03:01

- **Return on Compute:** A metric measuring how much business value is generated for every unit of processing power used.
- **Agentic Workflow:** A system where an AI agent doesn't just talk, but takes actions (like sending emails or updating databases) to complete a goal.
- **Red Teaming:** The practice of rigorously testing a system by attacking it to find security holes or biases.
- **Model Routing:** A system that looks at a user's request and decides which AI model is "smart enough" to answer it at the lowest cost.
- **Reverse Federalism:** When smaller government entities (like states) create laws that eventually become the standard for the entire country.
- **Diffusers:** A type of AI architecture used primarily for generating images and videos.
- **Prompt Injection:** A type of attack where a user tries to trick an AI into ignoring its safety rules.
