## Terminology

- **Red-teaming:** The practice of rigorously testing a system’s security and safety by simulating adversarial attacks.
- **Model Routing:** The logic used to determine which specific AI model is best suited to handle a given user request based on cost and complexity.
- **Fine-tuning:** Adjusting a pre-trained model on a smaller, specific dataset to improve its performance for a particular task or style.
- **Agentic AI:** AI systems capable of using tools and taking multi-step actions to achieve a specific goal with minimal human intervention.
- **Diffusers:** A library used for state-of-the-art diffusion models (like Stable Diffusion) for generating images and video.
- **Token Exposure:** A security risk where sensitive access keys (tokens) are inadvertently revealed, potentially allowing unauthorized access to data or compute.
